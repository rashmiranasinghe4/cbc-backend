🛠️ Cosmetics E-commerce Backend

A robust backend system for a cosmetics e-commerce platform, built using Node.js and Express.js. It manages authentication, product data, and order processing with secure and scalable architecture.

🚀 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

✨ Features

- 🔐 Secure Authentication (JWT + Google OAuth)
- 👥 Role-Based Access Control (Admin/User)
- 📦 Product Management (CRUD operations)
- 🧾 Order Processing System
- 🔒 Protected API Routes
- ⚙️ Middleware-based validation

🔗 API Architecture

- RESTful API design
- Handles all client-server communication
- Token-based authentication system

🌐 Deployment

- Hosted on Render
- Environment variables used for secure configuration

🗄️ Database

- MongoDB Atlas (Cloud-based database)
- Efficient data storage and retrieval

⚙️ Setup Instructions

1. Clone the repository
2. Install dependencies
   npm install
3. Configure environment variables
4. Run the server
   npm start
